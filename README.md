# Restaurant Reviews Sentiment Analysis (NLP)

## 🚀 Overview

This project focuses on **Natural Language Processing (NLP)** to analyze customer reviews of restaurants and classify them as **positive or negative sentiment**.

It demonstrates how text data can be preprocessed and used to build machine learning models for sentiment analysis.

---

## 🧠 Techniques Used

* Text Cleaning (removing punctuation, stopwords)
* Tokenization
* Stemming / Lemmatization
* Bag of Words (BoW) / TF-IDF
* Machine Learning Classification

---

## 📁 Project Structure

* `dataset/` → Restaurant_Reviews.tsv
* `nlp_model.py` → Main model implementation
* `notebooks/` → Jupyter notebooks (optional)

---

## 📌 Problem Statement

Analyze customer reviews and predict whether the sentiment is:

* Positive 😊
* Negative 😞
